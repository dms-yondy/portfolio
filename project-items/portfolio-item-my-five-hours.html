Project: My Five Hours

Description

This project was inspired by the idea of the five-hour rule. The five-hour rule says that we should spend one hour every weekday to learn something new. The something can be anything from learning about new technologies to learning more about an event in history. By making a little time each day to learn something new we can improve ourselves and prepare ourselves for the challenges we face day to day.

This project was written using the MERN stack, which stands for MongoDB, Express, React and Node.js

Technologies Used

JavaScript, HTML, CSS, React, Node.js and MongoDB
